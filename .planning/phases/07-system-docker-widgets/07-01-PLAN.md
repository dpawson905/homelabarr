  <action>
Install the systeminformation package: `npm install systeminformation`

Create `app/api/widgets/system-stats/route.ts` with a GET handler.

The route takes no query parameters -- it returns the current system snapshot.

Handler logic:
1. Import `si` from systeminformation:
   ```typescript
   import si from "systeminformation"
   ```
2. Fetch these metrics in parallel using `Promise.all`:
   - `si.currentLoad()` -- for CPU usage percentage
   - `si.mem()` -- for RAM used/total
   - `si.fsSize()` -- for disk usage (aggregate all mounted filesystems)
   - `si.networkStats()` -- for network throughput (rx/tx bytes per second)
3. Transform into a clean response shape:
   ```typescript
   {
     cpu: {
       usage: currentLoad.currentLoad,           // overall CPU usage %
       cores: currentLoad.cpus.map(c => c.load)  // per-core usage %
     },
     memory: {
       used: mem.used,      // bytes
       total: mem.total,    // bytes
       usage: (mem.used / mem.total) * 100  // percentage
     },
     disk: {
       used: totalDiskUsed,   // bytes (sum across filesystems)
       total: totalDiskSize,  // bytes (sum across filesystems)
       usage: (totalDiskUsed / totalDiskSize) * 100,
       filesystems: fsSize.map(fs => ({
         mount: fs.mount,
         used: fs.used,
         size: fs.size,
         usage: fs.use
       }))
     },
     network: {
       rx_sec: totalRxSec,    // bytes/sec (sum across interfaces)
       tx_sec: totalTxSec     // bytes/sec (sum across interfaces)
     }
   }
   ```
4. For disk: sum `used` and `size` across all entries from `si.fsSize()`. Filter out any filesystem where `size === 0` to exclude pseudo-filesystems.
5. For network: sum `rx_sec` and `tx_sec` across all entries from `si.networkStats()`. Filter out loopback interfaces (where `iface === "lo"` or similar).
6. Return as JSON with `Cache-Control: no-store` header (data must be fresh every request).
7. Wrap everything in try/catch, return 500 with `{ error: "Failed to collect system stats" }` on failure.

IMPORTANT: systeminformation functions are async and return Promises. All calls must be awaited.

IMPORTANT: On first call, `si.networkStats()` may return 0 for rx_sec/tx_sec because it needs a previous measurement. This is expected behavior -- the second poll will show real values. Do not try to work around this.

**Network Section:**

- Use arrow symbols (↓ for download/rx, ↑ for upload/tx)

Format helper function for bytes:
```typescript
function formatBytes(bytes: number, decimals = 1): string {
  if (bytes === 0) return "0 B"
  const k = 1024
  const sizes = ["B", "KB", "MB", "GB", "TB"]
  const i = Math.floor(Math.log(bytes) / Math.log(k))
  return `${(bytes / Math.pow(k, i)).toFixed(decimals)} ${sizes[i]}`
}
```
