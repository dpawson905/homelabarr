Create `app/api/auth/password/route.ts`:

PUT handler — change password:
- Accept `{ currentPassword: string, newPassword: string, confirmPassword: string }`
- Call `await getSessionFromCookies()` — if null, return 401 `{ error: "Unauthorized" }`
- Get the current hash from settings table
- Call `verifyPassword(currentPassword, hash)` — if false, return 400 `{ error: "Current password is incorrect" }`
- Validate newPassword.length >= 8
- Validate newPassword === confirmPassword
- Hash the new password and update the settings table: use `db.update(settings).set({ value: newHash, updatedAt: ... }).where(eq(settings.key, "passwordHash")).run()`
- Return 200 `{ success: true }`
