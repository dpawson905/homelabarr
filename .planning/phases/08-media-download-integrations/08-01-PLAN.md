<task type="auto">
  <name>Task 2: Build the shared service client</name>
  <files>lib/services/service-client.ts</files>
  <action>
Create `lib/services/service-client.ts` with two exported functions.

**Function 1: `resolveSecret(secretName: string): string | null`**

Synchronous function that:
1. Imports `db` from `@/lib/db` and `secrets` from `@/lib/db/schema` and `decrypt` from `@/lib/crypto/secrets`
2. Queries the secrets table by name: `db.select().from(secrets).where(eq(secrets.name, secretName)).get()`
3. If no secret found, returns `null`
4. Decrypts and returns the plaintext value: `decrypt(secret.encryptedValue, secret.iv, secret.authTag)`
5. Wrap in try/catch -- return `null` on decryption failure

**Function 2: `fetchService<T>(options: ServiceFetchOptions): Promise<ServiceResponse<T>>`**

Async function that makes an authenticated HTTP request to an external service:

1. Build the full URL: `new URL(options.endpoint, options.baseUrl)`. Make sure to handle trailing slashes on baseUrl. If baseUrl is `http://host:8989` and endpoint is `/api/v1/calendar`, result should be `http://host:8989/api/v1/calendar`. Use a helper: if endpoint starts with `/`, concatenate baseUrl.replace(/\/$/, "") + endpoint. Otherwise use new URL().

2. Add any queryParams to the URL's searchParams.

3. Build headers based on authType:
   - `"header-x-api-key"`: set `X-Api-Key: ${apiKey}`
   - `"header-x-plex-token"`: set `X-Plex-Token: ${apiKey}` and `Accept: application/json`
   - `"header-authorization"`: set `Authorization: MediaBrowser Token="${apiKey}"` (Jellyfin pattern)
   - `"query-apikey"`: append `apikey=${apiKey}` to URL searchParams (SABnzbd pattern). No special headers.

4. Make the fetch call with:
   - `method`: from options (default GET)
   - `headers`: as built above
   - `signal`: `AbortSignal.timeout(options.timeout ?? 10_000)`
   - `body`: if provided and method is not GET, `JSON.stringify(options.body)` with Content-Type application/json

5. If response is not ok, return `{ ok: false, error: "Service returned ${response.status}", status: response.status }`

6. Parse response as JSON and return `{ ok: true, data: parsed as T }`

7. Catch errors:
   - If error name is "TimeoutError" or "AbortError", return `{ ok: false, error: "Service request timed out" }`
   - Otherwise return `{ ok: false, error: "Failed to connect to service" }`

**Function 3: `getServiceConnection(config: Record<string, unknown> | null): { serviceUrl: string; apiKey: string } | null`**

Convenience function that:
1. Reads `serviceUrl` and `secretName` from config
2. Validates both are non-empty strings
3. Calls `resolveSecret(secretName)` to get the API key
4. If secret not found, returns `null`
5. Returns `{ serviceUrl, apiKey }`

This function is the one-liner that API routes will call at the top of every handler.
