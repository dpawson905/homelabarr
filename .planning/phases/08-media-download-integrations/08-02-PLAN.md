Build the Media Server widget that shows now-playing sessions and recently-added content from Plex or Jellyfin.

Purpose: Give homelab operators a live view of what is streaming on their media server and what content was recently added, directly from the dashboard.
Output: A server-side API proxy route and a client widget component supporting Plex and Jellyfin.

Infrastructure from Plan 01:
- `fetchService(options)` from lib/services/service-client.ts handles authenticated HTTP requests
- `getServiceConnection(config)` resolves serviceUrl + apiKey from widget config + secrets table
- ServiceAuthType: "header-x-plex-token" for Plex, "header-authorization" for Jellyfin

Plex API reference:
- Sessions (now playing): GET /status/sessions -- returns MediaContainer.Metadata array with Player, User info
- Recently added: GET /library/recentlyAdded -- returns MediaContainer.Metadata array
- Auth: X-Plex-Token header. Also needs Accept: application/json to get JSON instead of XML.
- Plex returns nested objects: each item has `type` (movie/episode/track), `title`, `grandparentTitle` (show name for episodes), `viewOffset` (current position ms), `duration` (total ms), `User.title`, `Player.state` (playing/paused/buffering)

Jellyfin API reference:
- Sessions: GET /Sessions -- returns array of session objects with NowPlayingItem, PlayState
- Recently added: GET /Items/Latest -- returns array of items (needs UserId query param, or use GET /Items?SortBy=DateCreated&SortOrder=Descending&Limit=20&Recursive=true&IncludeItemTypes=Movie,Episode,Audio)
- Auth: Authorization: MediaBrowser Token="key"
- Jellyfin sessions have: NowPlayingItem.Name, NowPlayingItem.SeriesName, PlayState.PositionTicks, NowPlayingItem.RunTimeTicks, UserName, PlayState.IsPaused

- Visual: rounded-lg border border-border bg-card, header with icon + title, overflow-hidden
- Icons: @hugeicons/react + @hugeicons/core-free-icons
- Sonner toast for error feedback

**Types file: `app/api/widgets/media-server/types.ts`**

Define the normalized response types that the widget component consumes (abstracted from Plex/Jellyfin differences):

```typescript
export interface NowPlayingItem {
  title: string           // movie title, episode title, or track title
  subtitle: string        // show name for episodes, artist for music, empty for movies
  user: string            // who is streaming
  state: "playing" | "paused" | "buffering"
  progressPercent: number // 0-100
  mediaType: "movie" | "episode" | "music" | "other"
  year?: number
  duration?: string       // formatted like "1h 23m"
}

export interface RecentlyAddedItem {
  title: string
  subtitle: string        // show name for episodes, artist for music
  mediaType: "movie" | "episode" | "music" | "other"
  addedAt: string         // ISO date string
  year?: number
}

export interface MediaServerResponse {
  nowPlaying: NowPlayingItem[]
  recentlyAdded: RecentlyAddedItem[]
  serverName?: string
}
```

**API route: `app/api/widgets/media-server/route.ts`**

Create a GET handler that:

1. Reads query params: `widgetId` (required)
2. Loads the widget config from DB using `getWidgetWithConfig(widgetId)` from `app/api/widgets/helpers.ts`
3. Extracts from config: `serviceUrl`, `secretName`, `serviceType` (one of "plex" | "jellyfin")
4. Uses `getServiceConnection(config)` from lib/services/service-client.ts to resolve the API key
5. If connection not resolved, return 400 with `{ error: "Service not configured" }`

**For Plex (`serviceType === "plex"`)**:
- Determine authType: `"header-x-plex-token"`
- Fetch now-playing: `fetchService({ baseUrl, apiKey, endpoint: "/status/sessions", authType })`
  - Plex response: `{ MediaContainer: { Metadata: [...] } }` (may be absent if nothing playing)
  - Map each session to NowPlayingItem:
    - title: `item.title`
    - subtitle: `item.grandparentTitle ?? ""` (show name for episodes)
    - user: `item.User?.title ?? "Unknown"`
    - state: map `item.Player?.state` ("playing"/"paused"/"buffering") to enum
    - progressPercent: `(item.viewOffset / item.duration) * 100` (both in ms)
    - mediaType: map `item.type` ("movie"/"episode"/"track") to enum
    - duration: format `item.duration` ms to "Xh Ym" string
- Fetch recently added: `fetchService({ baseUrl, apiKey, endpoint: "/library/recentlyAdded", authType, queryParams: { "X-Plex-Container-Size": "20" } })`
  - Map each to RecentlyAddedItem:
    - title: `item.title`
    - subtitle: `item.grandparentTitle ?? ""`
    - mediaType: map `item.type`
    - addedAt: convert `item.addedAt` (unix timestamp) to ISO string
    - year: `item.year`

**For Jellyfin (`serviceType === "jellyfin"`)**:
- Determine authType: `"header-authorization"`
- Fetch now-playing: `fetchService({ baseUrl, apiKey, endpoint: "/Sessions", authType })`
  - Jellyfin response: array of session objects, filter to those with `NowPlayingItem` present
  - Map each to NowPlayingItem:
    - title: `session.NowPlayingItem.Name`
    - subtitle: `session.NowPlayingItem.SeriesName ?? ""`
    - user: `session.UserName ?? "Unknown"`
    - state: `session.PlayState?.IsPaused ? "paused" : "playing"`
    - progressPercent: `(session.PlayState?.PositionTicks / session.NowPlayingItem.RunTimeTicks) * 100` (both in ticks)
    - mediaType: map `session.NowPlayingItem.Type` ("Movie"/"Episode"/"Audio") to enum (lowercase + normalize)
    - duration: format RunTimeTicks (ticks = 10M per second) to "Xh Ym"
- Fetch recently added: `fetchService({ baseUrl, apiKey, endpoint: "/Items/Latest", authType, queryParams: { Limit: "20", IncludeItemTypes: "Movie,Episode,Audio", Fields: "DateCreated" } })`
  - Map each to RecentlyAddedItem:
    - title: `item.Name`
    - subtitle: `item.SeriesName ?? ""`
    - mediaType: map `item.Type`
    - addedAt: `item.DateCreated ?? item.PremiereDate ?? ""`
    - year: `item.ProductionYear`

Return the combined MediaServerResponse with both arrays. If either sub-fetch fails, return an empty array for that section rather than failing the whole request. Cache-Control: no-store.

Add a helper to format milliseconds to duration string: `function formatDuration(ms: number): string` -- returns "1h 23m" format. For ticks (Jellyfin), divide by 10_000_000 first to get seconds.

function timeAgo(isoDate: string): string {
  const diff = Date.now() - new Date(isoDate).getTime()
  const minutes = Math.floor(diff / 60000)
  if (minutes < 60) return `${minutes}m ago`
  const hours = Math.floor(minutes / 60)
  if (hours < 24) return `${hours}h ago`
  const days = Math.floor(hours / 24)
  return `${days}d ago`
}
